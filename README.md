# 🛡️ Matemáticas que Resuelven Problemas: Fibonacci y Números Primos

Este proyecto corresponde al desarrollo de una aplicación web interactiva creada con HTML, CSS y JavaScript. La página implementa la sucesión de Fibonacci junto con la validación de números primos para simular un sistema conceptual de generación y filtrado de códigos seguros.

La aplicación fue diseñada utilizando un enfoque visual moderno con estilo Glassmorphism y diseño responsivo adaptable a dispositivos móviles, tablets y computadoras.

# 🎯 Problema Planteado

En los sistemas informáticos modernos, los patrones predecibles pueden representar vulnerabilidades de seguridad. Este proyecto propone una simulación matemática donde se generan secuencias numéricas dinámicas utilizando Fibonacci y posteriormente se identifican números primos dentro de la secuencia.

La idea consiste en representar cómo ciertos patrones matemáticos pueden utilizarse para procesos de validación, filtrado y generación de datos más difíciles de predecir.
